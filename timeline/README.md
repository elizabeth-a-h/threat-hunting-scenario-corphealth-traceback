# Timeline Summary — CorpHealth: Traceback Threat Hunt

This folder contains the investigation timeline for the CorpHealth: Traceback threat hunt. The timeline documents all key events, from initial suspicious activity to persistence and external command-and-control interactions.

## Contents

- `Full_Timeline.md` — Complete step-by-step chronology of attacker activity, including timestamps, accounts, processes, files, and network events.
- `Timeline_Condensed.md` — High-level version highlighting key events and stages, suitable for executive reporting or quick reference.

## Usage

Analysts and reviewers can use these timelines to:

- Understand the order of suspicious activity.
- Correlate alerts, telemetry, and investigative queries.
- Validate findings in relation to MITRE ATT&CK techniques.
- Provide documentation for escalation, reporting, or training purposes.
